About

Background

I'm a third-year Computer Science undergraduate researching explainable AI for medical imaging, and building production-grade ML and backend systems on the side — the kind with monitoring dashboards, CI/CD pipelines, and drift detection, not just a notebook that ran once.

Lately I keep gravitating toward the same problem from different angles: how do you trust a system you didn't watch reason? A model that explains its diagnosis with Grad-CAM, an LLM gateway that blocks what it shouldn't say, a RAG pipeline that catches its own hallucinations before they reach a user — same question, different systems.

I hold a published Indian patent and presented research at AiCCONs 2026. I'm heading into my final year, with internship experience spanning full-stack development, backend systems, and startup web engineering.

Real-Time Fraud Detection System
Repo
Catches fraud in real time — and knows when it isn't sure.

An XGBoost + autoencoder ensemble (0.984 ROC-AUC) scoring transactions for fraud across 284K records with extreme class imbalance, handled with SMOTE. Served through FastAPI, containerized, monitored live with Prometheus and Grafana, checked for drift with Evidently AI, and backed by a full CI/CD test suite on GitHub Actions.

Self-Healing RAG Pipeline
Repo
A RAG system that catches its own hallucinations.

Standard RAG retrieves and generates. This one adds a critic step: a LangGraph agent checks whether an answer is actually grounded in the retrieved documents, and if it isn't, reformulates the query and retries — up to twice — before admitting it doesn't know, instead of guessing.

Explainable AI for Colon Cancer Detection
AiCCONs 2026
Interprets its own diagnosis, not just makes one.

A deep learning framework spanning histopathology and endoscopy images, merging the LC25000 and Kvasir datasets into a single three-class classification task. Reached ~99% test accuracy with ResNet50, with Grad-CAM and SHAP layered on top so the model's reasoning stays visible to a clinician, not just its output.
